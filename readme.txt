== Description ==

**Show Widget by Logic** plugin which allows to operate by WordPress widgets visibility using "WordPress Conditional Tags" and "WooCommerce Conditional Tags" logic

Latest PHP 8.x - COMPATIBLE!

### "Show Widget by Logic" can:

&#9989;&nbsp;**Wordpress:** use [WordPress Conditional Tags](https://codex.wordpress.org/Conditional_Tags) to operate by site widgets visibility

&#9989;&nbsp;**WooCommerce:** use [WooCommerce Conditional Tags](https://woocommerce.com/document/conditional-tags/) to operate by woo shop widgets visibility

&#9989;&nbsp;**Custom:** use your own custom functions created in file functions.php of the current WordPress theme


== Installation ==
* Download to your plugin directory or simply install via WordPress admin interface.
* Activate.
* Use textarea on the bottom of each widget to set its logic if necessary
* Save
* Use.
